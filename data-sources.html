Data Sources (planned)

In this preview build, CryptoFeeScope uses fixed mock data only. The goal is to first stabilize the UI and logic. In later phases, the dashboard will fetch real values from public gas/fee APIs, primarily on free plans.

Candidate APIs

  • Bitcoin family: mempool.space API
  • Ethereum / BSC / Polygon: gasprice.io, Blocknative Gas Estimator
  • Solana / TRON / XRP: official endpoints or OKLink API
  • Multi-chain aggregators: DefiLlama API, GasFees and similar services

Once live data is wired in, each row will display not only the fee and speed, but also the underlying data source and the exact fetch timestamp, so users can see how fresh the values are.
