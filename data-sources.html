Data Sources

In this public demo, all values come from a CoinGecko Demo snapshot at /data/fee_snapshot_demo.json to avoid API rate limits. The goal is to stabilize the UX first, then plug into production-grade sources.

Current demo feed

  • CoinGecko Demo API (prices + fee hints) refreshed by GitHub Actions
  • Snapshot hosted on Cloudflare Pages for predictable caching
  • Client-side conversion between USD and JPY

Candidate APIs for production (planned)

  • Bitcoin family: mempool.space API
  • Ethereum / BSC / Polygon: gasprice.io, Blocknative Gas Estimator
  • Solana / TRON / XRP: official endpoints or OKLink API
  • Multi-chain aggregators: DefiLlama API, GasFees and similar services

Once live data is wired in, each row will show the upstream provider, fetch timestamp, and any rate-limit caveats so users can gauge freshness.
