About CryptoFeeScope

CryptoFeeScope is a lightweight dashboard to compare transaction fees and approximate confirmation times across major L1 and L2 networks on a single screen.

What it shows

  • Chain name (e.g. Bitcoin (L1), Arbitrum (L2 on ETH))
  • Estimated fee in USD (and converted to other fiat currencies like JPY)
  • Estimated time to first confirmation
  • Simple status label: Fast / Avg / Slow
  • Timestamp of the last update (‚ÄúUpdated‚Äù column)

How the numbers are intended to work

  • Fee: base gas/fee estimate per chain, expressed in USD, then converted to the selected fiat currency on the client.
  • Speed: a rough estimate based on block time and typical confirmations, shown as a reference value, not a guarantee.
  • L2 networks are labeled with their base L1, e.g. Arbitrum (L2 on ETH), Optimism (L2 on ETH).

In the current preview build, all values come from mock data for UI testing. A later phase will switch to live data from public gas/fee APIs.
