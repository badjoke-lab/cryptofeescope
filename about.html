About CryptoFeeScope

CryptoFeeScope is a lightweight dashboard to compare transaction fees and approximate confirmation times across major L1 and L2 networks on a single screen. The MVP uses the CoinGecko Demo snapshot hosted on Cloudflare Pages to keep the UI fast and predictable.

What it currently shows

  • Chain name (e.g. Bitcoin (L1), Arbitrum (L2 on ETH))
  • Estimated fee in USD and JPY with compact formatting
  • Estimated time to first confirmation
  • Simple status label: Fast / Avg / Slow
  • Timestamp of the last update pulled from the snapshot JSON

How the numbers are intended to work

  • Fee: base gas/fee estimate per chain expressed in USD, then converted to the selected fiat currency on the client.
  • Speed: a rough estimate based on block time and typical confirmations, shown as a reference value, not a guarantee.
  • L2 networks are labeled with their base L1, e.g. Arbitrum (L2 on ETH), Optimism (L2 on ETH).

Planned additions

  • Historical trend lines and more fiat currencies (EUR is first in line)
  • Live Worker-backed API that no longer relies on demo endpoints
  • Per-row data provenance with source labels and fetch timestamps

The current build is powered by demo data for stability while UI polish completes. Production-grade data sources and additional UX improvements are planned in the next phases.
