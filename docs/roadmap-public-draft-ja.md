## 🔧 CryptoFeeScope Roadmap / ロードマップ

> ※あくまで「開発予定」です。仕様や順番は、API の状況や開発リソースにより変更される場合があります。

---

### ✅ Phase 1 — Live Snapshot Dashboard（現在値ダッシュボード）

**Status:** In progress → Public MVP

- 各チェーンの **現在の平均送金手数料** と **目安の処理速度** を一覧表示
- USD / JPY 切り替え
- チェーンごとのステータス（fast / normal / slow）
- ダーク / ライトテーマ切り替え

**Goal:**  
「今どのチェーンがどれくらい安い／高いか」を一目で確認できる“瞬間スナップショット版”の完成。

---

### 🔜 Phase 1.5 — UI polish & 24h change（UI調整＋24時間変化）

**Status:** Planned

- テーブルの見やすさ改善（モバイル / PC 両対応）
- 24時間の価格変化率（例：+3% / -2%）を追加表示  
  → “今だけ”でなく「直近の方向感」もわかるようにする
- 公開用ドキュメント整備（更新方法・データ仕様の簡易説明）

**Goal:**  
瞬間値だけのツールから一歩進めて、**「今日の相場感」もざっくり分かる軽量ダッシュボード**にする。

---

### 🚀 Phase 2 — Common API v1（共通API v1 / Worker）

**Status:** Planned

- Cloudflare Workers 上に **共通API** を構築
  - `GET /v1/fees/current` で現在のスナップショットを返す
  - レスポンス形式を統一：`{ ok, ts, cacheSec, data, error, source }`
  - 外部API（CoinGecko 等）へのアクセスをサーバー側でキャッシュ
- CryptoFeeScope フロントエンドは **共通APIからデータ取得**する構成に移行
- 将来的に **WCWD や他ツールも同じAPIを再利用**できる基盤にする

**Goal:**  
ブラウザ直叩きに依存しない、**壊れにくいデータ基盤**を用意する。

---

### 🐦 Phase 3 — X Bot v1（シンプル投稿ボット）

**Status:** Planned

- 共通API（`/v1/fees/current`）からデータを取得して、
  - 1日1回～数回程度、X アカウントから自動投稿（または半自動）
  - 例：  
    - 「いま一番安いチェーン TOP3」  
    - 「ETH/BTC の現在手数料と24h変化」
- 初期バージョンは **テキストのみのシンプル投稿** を想定  
  （画像生成や高度な分析は後のフェーズ）

**Goal:**  
サイトを開かなくても、**Xのタイムライン上でざっくり手数料状況が分かる**ようにする。

---

### 📈 Phase 4 — History & Charts（履歴とグラフ表示）

**Status:** Future

- 一定間隔でスナップショットを保存し、**数時間〜数日分の履歴**を表示
- 期間別の簡易グラフ（24h / 7d / 30d）  
  → チェーンごとの「平均的な安さ」「混雑しやすさ」を可視化
- 将来的には共通API側に `/v1/fees/history` を追加し、  
  他ツールや Bot からも履歴データを再利用できる形に拡張

**Goal:**  
「今」だけでなく、**“どのチェーンが長期的に安定して安いか”** を比較できるツールへ育てる。

---

## 💰 Support / 寄付について

CryptoFeeScope は、**個人運営の無料プロジェクト**として開発・運用されています。  
寄付は、以下のような目的に優先的に使われます：

- 外部API（例：CoinGecko）の有料プラン・上位プランへのアップグレード  
- Cloudflare / X Bot などのインフラ維持・モニタリングコスト
- 新機能（共通API・履歴・Bot機能など）の開発時間の確保

> 「このロードマップの先も見てみたい」「長く続いてほしい」と思ったら、  
> サイト内の寄付リンクからサポートしてもらえると、とても助かります 🙏
